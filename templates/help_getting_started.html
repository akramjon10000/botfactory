Muvaffaqiyatli Start Qo'llanmasi

BotFactory AI orqali biznesingizni avtomatlashtirish qadamlari — oddiy, tushunarli va tezkor yo'riqnoma.

Bosh sahifa

1. Ro'yxatdan o'tish va kirish

  1. Ro'yxatdan o'tish sahifasidan hisob yarating.
  2. Elektron pochta/username va parol bilan kiring.

2. Bot yaratish

  1. Boshqaruv paneliga o'ting.
  2. "Yangi bot yaratish" tugmasini bosing.
  3. Bot nomini kiriting va Telegram Bot tokenini qo'shing.
Hozirda faqat Telegram platformasi qo'llab-quvvatlanadi. Boshqa platformalar kelajakda qo'shiladi.

3. Telegram sozlash (tez start)

  1. @BotFather ga kiring → /newbot buyrug'ini yuboring.
  2. Bot uchun nom va username tanlang (masalan: doys_bot).
  3. BotFather bergan tokenni nusxalab, bot yaratish sahifasiga kiriting.
  4. Botga /start yozing — javob keldi, demak u ishlayapti!
Eslatma: Biz polling + deduplikatsiya ishlatamiz. Markaziy bot_manager orqali start/stop boshqariladi.

4. Bilimlar bazasi (Knowledge Base)

Botingizni aqlli qilish uchun bilim bazasiga ma'lumotlarni qo'shing. Bot tahrir sahifasida 5 ta tab mavjud:

📁 Fayl
.txt, .csv, .docx, .xlsx
📝 Matn
Erkin matn yozish
❓ Q&A
Savol-javob juftliklari
📦 Mahsulot
Nom, narx, tavsif, rasm
📞 Aloqa
Telefon, manzil, ish vaqti
AI chatbot dialoglar paytida bilim bazasidagi ma'lumotlardan avtomatik foydalanadi.

5. MiniApp (Telegram Web App)

Har bir bot uchun avtomatik ravishda Telegram Web App yaratiladi. MiniApp quyidagilarni o'z ichiga oladi:

  • 📦 Katalog — mahsulotlar/xizmatlar ro'yxati (qidiruv va saralash bilan)
  • 🤖 AI Chat — sun'iy intellekt yordamchisi (matnli va ovozli)
  • 🛒 Savat — buyurtma berish tizimi
  • 📞 Aloqa — biznes kontakt ma'lumotlari
MiniApp ranglarini (aksent, fon, kartochka) bot tahrir sahifasidagi "MiniApp Tashqi Ko'rinishi" bo'limidan sozlash mumkin.

6. Ovozli chat (Premium)

Premium foydalanuvchilar uchun ovozli chat xususiyati mavjud:

  • MiniApp'da 🎤 tugmani bosib ovozli xabar yuboring.
  • AI ovozni matnga o'girib, bilim bazasidan javob topadi.
  • Gemini Native Audio modeli orqali yuqori sifatli transkripsiya.
Bu xususiyat faqat Premium obunachilarga mavjud. Obunani boshqaruv panelidan yangilang.

7. Xabar yuborish (individual/broadcast)

  1. Dashboard → bot yonidagi "Xabar yuborish" bo'limiga kiring.
  2. Mijozni tanlang yoki "Barchaga yuborish" ni belgilang.
  3. Matnni kiriting va "Yuborish" tugmasini bosing.

8. Suhbat kuzatuvi (Monitoring)

  • Bot yaratishda yoki tahrirlashda "Suhbat xabarnomalarini yoqish" ni belgilang.
  • Admin Chat ID — sizning shaxsiy Telegram ID raqamingiz.
  • Bildirishnoma kanali — shaxsiy kanal yoki guruh.

Mijozlar botga yozganda, barcha suhbatlar sizning Telegram'ingizga yoki ko'rsatilgan kanal/guruhga yuboriladi.

9. Xavfsizlik

  • Tokenlaringiz shifrlangan holda xavfsiz saqlanadi.
  • Admin panelga faqat ro'yxatdan o'tgan foydalanuvchilar kiradi.
  • API kalitlari server tomonida himoyalangan.
